# B7-H3(CD276)靶点详细介绍

> 生成时间:2026-07-12
> 用途:爸爸(张建来,Ⅳ 期 mPDAC)未来治疗靶点评估参考

---

## 一、B7-H3 是什么?

### 基本信息
- **全称:** B7 Homolog 3(B7 同源家族第 3 成员)
- **别名:** CD276
- **家族:** B7 免疫检查点家族(PD-L1、CTLA-4 都属这个家族)
- **发现时间:** 2001 年(Chapoval 等)
- **本质:** 一种跨膜糖蛋白,分子量约 45-70 kDa
- **正常表达:** 正常组织表达极低(几乎不表达)
- **肿瘤表达:** 在多种实体瘤中高表达,是理想的肿瘤特异性靶点

### 双重生物学功能 ⭐

| 功能 | 机制 |
|------|------|
| **免疫抑制** | 抑制 T 细胞增殖和细胞因子分泌(与 PD-L1 类似,但机制不同) |
| **促肿瘤生长** | 直接促进肿瘤细胞增殖、迁移、侵袭 |
| **促血管生成** | 支持肿瘤微环境血管新生 |
| **化疗耐药** | 促进 EMT(上皮间质转化),导致耐药 |

**核心特点:B7-H3 同时是"免疫检查点"和"肿瘤直接靶点"两重身份。**

## 二、B7-H3 在 PDAC(胰腺癌)中的意义

### ⭐⭐ 高表达率

| 肿瘤类型 | B7-H3 阳性率 | 强阳性(3+) |
|---------|-------------|-------------|
| **胰腺癌(PDAC)** | **~93%** | **60-70%** |
| 胆管癌 | ~85% | ~50% |
| 非小细胞肺癌 | ~80% | ~50% |
| 前列腺癌 | ~93% | ~80% |
| 神经母细胞瘤 | >95% | 极高 |
| 卵巢癌 | ~90% | ~55% |

**PDAC 是 B7-H3 表达率最高的癌种之一!**

### 与预后关系
- B7-H3 高表达 → 总生存期 OS 显著缩短(HR ~1.8-2.5)
- 与转移相关(尤其肝转移)
- 与化疗耐药相关(吉西他滨、氟尿嘧啶)
- **爸爸的情况:Ⅳ 期 mPDAC + 肝转移,B7-H3 大概率高表达**(需活检验证)

## 八、B7-H3 的局限与风险 ⚠️

1. **单药疗效有限** — 需要与 ADC 载荷或 CAR-T 结合才能发挥
2. **PDAC 单独队列数据少** — 目前多在实体瘤篮子试验中入组
3. **ADC 毒性** — DXd 载荷可能引起:间质性肺炎(ILD)、血液学毒性、消化道反应
4. **异质性** — 肿瘤内表达不均匀,可能有耐药亚群

> B7-H3 是 PDAC 领域除 Claudin 18.2 之外最有希望的新靶点,尤其在 ADC 药物开发上进展快速。爸爸的组织样本可以补做 IHC 检测,如高表达 2+/3+,可考虑入组 HS-20093 / DS-7300 / YL201 等 Ⅱ 期临床试验。
