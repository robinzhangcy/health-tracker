#### 临床意义
- 三腔管 = 胃肠减压 + 空肠营养通路 + 备用通道
- 通常包括：
  1. 胃腔（胃肠减压、引流胃内容物）
  2. 空肠腔（经空肠肠内营养通道，跨过十二指肠梗阻段）
  3. 球囊/给药腔
- **针对父亲的情况**：完美匹配
  - 胰头癌侵犯十二指肠降段+水平段 → 机械性梗阻
  - 6/21 反酸呕吐 24h → 胃排空障碍
  - 胆道支架在位但十二指肠仍受压
  - 空肠营养可绕过梗阻段，维持肠内营养

#### 为什么这是好消息 ⭐
1. **绕过十二指肠梗阻，恢复肠内营养** — 比全肠外营养(TPN)安全
2. **胃肠减压** — 缓解反酸呕吐症状
3. **维持肠道屏障功能** — 减少肠道菌群移位（本次胆管炎诱因之一）
4. **为后续化疗提供营养基础** — Ⅳ 期患者营养是预后关键
5. **可能避免/推迟十二指肠金属支架** — 先观察症状
